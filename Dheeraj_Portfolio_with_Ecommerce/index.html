About

Backend-focused, detail-obsessed.

I'm an MCA graduate passionate about Java backend development — the layer of a product most users never see, but every user depends on.

My day-to-day toolkit is Spring Boot, Servlets, JSP and MySQL, and I've deployed and hosted applications on Azure using IIS. I care about clean data models, predictable APIs, and code that's easy for the next developer to read.

Outside of coursework projects, I build full working demos — not mockups — so you can actually click through a hospital management system, a news portal, or a shopping cart and see the backend behind it.
